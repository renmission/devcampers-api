# Devcampers API v1.0

> Backend API for Devcampers application.

## Usage

Rename "config/config.env.env" to "config/config.env" and update the values/settings.

## Install dependencies
```
npm install
```

## Run App
```
# Run in dev mode
npm run dev

# Run in prod mode
npm start
```
